# Coerce to GDALG

Coerce to GDALG

## Usage

``` r
as_gdalg(x, ..., call = rlang::caller_env())
```

## Arguments

- x:

  An object to coerce to class `"gdalg"`.

- ...:

  Additional arguments passed to specific methods.

- call:

  The calling environment, used for error messages.

## Value

An object of class `"gdalg"`.
