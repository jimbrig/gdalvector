# Sync Between VSI Paths

Sync Between VSI Paths

## Usage

``` r
vsi_sync(src, dst, ...)
```

## Arguments

- src:

  Source path.

- dst:

  Destination path.

- ...:

  Passed to
  [`gdalraster::vsi_sync()`](https://firelab.github.io/gdalraster/reference/vsi_sync.html).

## Value

Invisibly returns the sync result from GDAL.
