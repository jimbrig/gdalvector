
[GDAL Vector Drivers](https://gdal.org/en/stable/drivers/vector/index.html):

## Supported "Read" Drivers

**Note**: The following are considered "first class read" drivers, or "traditional" formats used for geospatial vector data sources. These formats are typically used as input read sources.

- [GeoJSON](https://gdal.org/en/stable/drivers/vector/geojson.html)
- [GeoPackage ("GPKG")](https://gdal.org/en/stable/drivers/vector/gpkg.html)
- [ESRI Shapefile / DBF](https://gdal.org/en/stable/drivers/vector/shapefile.html)
- [ESRI File Geodatabase (OpenFileGDB)](https://gdal.org/en/stable/drivers/vector/openfilegdb.html)

## Supported "Write" Drivers

**Note**: The following are considered "first class write" drivers, or "modern" and "cloud-native" formats used for geospatial vector data. These formats are typically used as output write targets.

- [FlatGeobuf (FGB)](https://gdal.org/en/stable/drivers/vector/flatgeobuf.html)
- [(Geo)Parquet](https://gdal.org/en/stable/drivers/vector/parquet.html)
- [PMTiles](https://gdal.org/en/stable/drivers/vector/pmtiles.html)

## Other Supported Drivers

**Note**: The following are considered "other" drivers, which we consider to be relevant and useful, but do not have first class, dedicated function support in `gdalvector`.

- [ADBC](https://gdal.org/en/stable/drivers/vector/adbc.html)
- [Arrow](https://gdal.org/en/stable/drivers/vector/arrow.html)
- [CSV](https://gdal.org/en/stable/drivers/vector/csv.html)
- [MEM: In Memory Datasets](https://gdal.org/en/stable/drivers/vector/mem.html)
- [GDALG: GDAL Streamed Algorithm](https://gdal.org/en/stable/drivers/vector/gdalg.html)
- [SQLite / Spatialite RDBMS](https://gdal.org/en/stable/drivers/vector/sqlite.html)
- [VRT: OGR Virtual Format](https://gdal.org/en/stable/drivers/vector/ogr_vrt.html)
- [OGC WFS Service](https://gdal.org/en/stable/drivers/vector/wfs.html)
- [ESRIJSON / FeatureService Driver](https://gdal.org/en/stable/drivers/vector/esrijson.html)
- [GeoJSONSeq: Sequence of GeoJSON Features](https://gdal.org/en/stable/drivers/vector/geojsonseq.html)
- [MVT: Mapbox Vector Tiles](https://gdal.org/en/stable/drivers/vector/mvt.html)
- [PostgreSQL / PostGIS](https://gdal.org/en/stable/drivers/vector/pg.html)
- [TopoJSON](https://gdal.org/en/stable/drivers/vector/topojson.html)
- [MS Office Open XML Spreadsheet](https://gdal.org/en/stable/drivers/vector/xlsx.html)
